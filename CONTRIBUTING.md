# Contributing

Issues and pull requests are welcome. Please keep reports and changes clear, focused, and easy to verify.

## Issues

When opening an issue, include:

- What you were trying to do
- What you expected to happen
- What actually happened
- Steps to reproduce the problem
- Your operating system
- The AI app you used
- Relevant error output or logs

For installer, update, preview, sync, or browser inspection issues, also include the command you ran and whether this was a fresh install, reinstall, or update.

## Pull Requests

Before opening a pull request:

- Keep the change focused on one problem or feature
- Explain why the change is needed
- Include a short summary of what changed
- Run the relevant checks and include the results
- Mention platform notes for macOS or Windows when installer/runtime behavior changes

For local setup, development workflow, and release details, see [DEVELOPMENT.md](DEVELOPMENT.md).
